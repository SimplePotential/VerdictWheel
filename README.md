# Weighted Spinning Wheel

Simple vanilla HTML/CSS/JS demo that draws a wheel with wedges sized by provided weights and spins to a weighted random result.

How to run

1. Open `index.html` in a browser (double-click or serve from a simple static server).
2. Add options (label + weight), then click `Spin`.

Files

- `index.html` — main page
- `style.css` — styles
- `script.js` — logic: draw wheel, pick weighted winner, animate spin
